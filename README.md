## Google Ads Scripts

### budget-pacing.js
Monitor daily budget pacing across campaigns. Alerts on 20%+ deviation.

### search-term-audit.js
Weekly search term audit. Flags wasted spend and missed keyword opportunities.

### quality-score-monitor.js
Tracks Quality Score changes. Alerts on drops >2 points.
